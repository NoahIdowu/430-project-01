1. Get All Pokémon

Relative URL: /api/pokemon

Supported Methods: GET, HEAD

Query Params:

  • type: Filters by Pokémon type.

Returns: JSON Array of Pokémon objects.

Example Usage: /api/pokemon?type=fire&limit=5

2. Add a Pokémon

Relative URL: /api/pokemon

Supported Methods: POST

Body Params:

  • name: The name of the Pokémon.
  • type: The type of the Pokémon.

Returns: JSON Object of the created Pokémon.

Example Usage: Send JSON body {"name": "Mewthree", "type": "Psychic"}
